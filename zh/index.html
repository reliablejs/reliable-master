Reliable

Macaca 的持续交付服务套件。

使用文档

Macaca 无缝融合

与 Macaca 工具链生态无缝融合，与 Mock、端到端测试、报告器、覆盖率等能力无缝结合，完成 pipeline 闭环。

Docker 部署

支持 Docker 在内的多种部署方式，开箱即用的服务部署体验助你快速搞定自己的服务。

集成 CI/CD Runner

支持主流的研发流程软件 Gitlab CI 和 Jenkins 的集成使用，上报客户端灵活对接。

适用场景

Reliable 适合在持续集成阶段与现有研发流程打通和衔接，当然也可以独立使用。
